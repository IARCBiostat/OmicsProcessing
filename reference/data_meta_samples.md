# Simulated Dataframe 3

A dataframe with ID, batch, sex, and age information.

## Usage

``` r
data(data_meta_samples)
```

## Format

A data frame with 100 rows and 4 variables:

- ID_sample:

  ID from ID_sample column in data

- batch:

  randomly assigned levels (N = 10)

- sex:

  randomly assigned 1 or 2

- age:

  mean 55; SD 10
