# Finalize kept columns

Always keeps the non-target columns alongside the retained target
columns.

## Usage

``` r
final_cols(target_cols_kept, non_target_cols, all_names)
```

## Arguments

- target_cols_kept:

  Character vector of retained target columns.

- non_target_cols:

  Character vector of non-target column names.

- all_names:

  Character vector of all column names in the dataset.

## Value

Character vector of final kept column names.
